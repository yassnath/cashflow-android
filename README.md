# Tabungan Solvix Studio (Android)

Aplikasi pencatatan keuangan harian dengan fokus pengalaman mobile yang ringkas, aman, dan mudah dipakai. Cocok untuk mencatat pemasukkan, pengeluaran, target, hingga laporan, dengan tampilan yang bisa disesuaikan.

## Keunggulan
- Mobile-first UI: layout rapi dan nyaman di layar kecil.
- Multi tema: pilihan tema standar dan kartun sesuai mood.
- Ringkasan cepat: pemasukkan, pengeluaran, saldo tampil instan.
- Laporan visual: grafik bulanan/tahunan + export PDF/CSV.
- Proteksi privasi: layar onboarding/login menutupi konten aplikasi.
- Akses cepat: opsi login fingerprint (jika perangkat mendukung).

## Fitur Utama
- CRUD pemasukkan & pengeluaran
- History transaksi + filter rentang tanggal
- Target/Dreams tracker
- Kalkulator
- Profil & pengaturan
- Bahasa Indonesia / English

## Cara Menjalankan
1) Buka project Android di Android Studio.  
2) Sync Gradle, lalu pilih device/emulator.  
3) Jalankan konfigurasi `app` (Debug).
